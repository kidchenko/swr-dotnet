# ISwr.PutAsync&lt;T&gt; method

Sends a PUT request, deserializes the response, and invalidates the URL's cache entry plus any additional keys.

```csharp
public Task<T?> PutAsync<T>(string url, object content, params string[] additionalInvalidateKeys)
```

| parameter | description |
| --- | --- |
| T | The type of the response body to deserialize. |
| url | The URL to PUT to. Also used as the primary cache key to invalidate. |
| content | The request body object, serialized as JSON. |
| additionalInvalidateKeys | Additional cache keys to evict after the request succeeds. |

## Return Value

The deserialized response body, or null if the response body is empty.
