# ISwr.GetAsync&lt;T&gt; method

Fetches data and awaits until the initial data is available. Use in Blazor's `OnInitializedAsync` to ensure the component renders with data on first paint.

```csharp
public Task<SwrResult<T>> GetAsync<T>(string key, SwrOptions? options = null)
```

| parameter | description |
| --- | --- |
| T | The type of data to fetch and cache. |
| key | The URL or cache key identifying this data. |
| options | Per-request override of global SWR options, or null to use defaults. |

## Return Value

A task that completes with a [`SwrResult`](../SwrResult-1.md) once initial data is available.
