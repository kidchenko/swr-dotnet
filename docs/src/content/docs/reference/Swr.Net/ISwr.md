# ISwr interface

Provides stale-while-revalidate caching for HTTP data fetching. Register via [`AddSwrForBlazor`](../Swr.Net.Extensions/SwrServiceCollectionExtensions/AddSwrForBlazor.md) or [`AddSwrForAspNetCore`](../Swr.Net.Extensions/SwrServiceCollectionExtensions/AddSwrForAspNetCore.md).
